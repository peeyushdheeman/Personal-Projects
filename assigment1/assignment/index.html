the font-family

A font family is a set of fonts that have a common design.
Fonts within a family, however, differ from each other in
style such as the weight (light, normal, bold, semi-bold, etc.) and
the (roman or upright, italic and oblique) slant . An example of
a font family is Times New Roman, which consists of a roman, italic,
bold and bold italic version of the same typeface.
